## React Router v6 Cannot Get URL

사실 v6 버전인지는 중요치 않았던 Error

Router를 사용하는데 자꾸 Match 되지 않는다고...

단언컨데 오늘 최고의 플레이
Webpact config

historyApiFallback 속성에 대해서 알아보고 정리하기.

```
devServer: {
  historyApiFallback: true
},
```
